OptiHK 招聘

一起构建光子硬件技术栈。

我们正在招聘能够跨越实验室验证、产品限制与客户交付的实战型工程师与管理人才。

全职职位
高级工程师 / 工程师，电子 IC 设计

为 OptiHK 光引擎设计电子 IC 模块与接口，连接高速电子与集成光子平台。

薪酬

HKD 390k ~ 700k

工作内容

  • 设计 ASIC、混合信号、驱动器、接收器、控制与监测电路。
  • 支持 PIC-EIC 集成、原型启动与测量调试。
  • 与光子、封装、板级与测试工程师协作推进产品周期。

任职要求

  • 具电气工程、微电子、IC 设计或相关学科学位。
  • 具 ASIC、RF、混合信号、高速或光电子 IC 设计经验者优先。
  • 具清晰文件撰写与跨职能工程沟通能力。
高级工程师 / 工程师，光子 IC 设计

为紧凑型光引擎与传感系统设计集成光子电路与硅光子基础模块。

薪酬

HKD 390k ~ 700k

工作内容

  • 设计 PIC 版图、波导、耦合器、分束器、滤波器与路由结构。
  • 针对封装、测试与系统级限制优化光子元件。
  • 与 EIC 与封装团队协同设计光电元件。

任职要求

  • 具光子学、光学工程、电气工程、物理或相关学科学位。
  • 具硅光子或集成光子元件设计经验者优先。
  • 能将仿真与测量反馈转化为设计更新。
高级工程师 / 工程师，嵌入式电子设计

开发嵌入式电子、板级控制与测量支持系统，用于光电子引擎验证。

薪酬

HKD 390k ~ 700k

工作内容

  • 设计嵌入式控制板、固件接口、偏压、监测与测试电路。
  • 支持实验室启动、调试、自动化与可重复测量流程。
  • 协调电气设计与 PIC、EIC、封装及软件需求。

任职要求

  • 具电气工程、电子、嵌入式系统或相关学科学位。
  • 具板级设计、固件、仪器控制或实验室调试经验者优先。
  • 能以清晰纪录与数据处理原型硬件问题。
高级工程师，先进封装

负责 PIC/EIC 集成、热控制、光学对准与可靠组装的先进封装设计及流程协调。

薪酬

HKD 390k ~ 700k

工作内容

  • 设计集成光子 IC、电子 IC、板卡与光接口的封装。
  • 解决热、机械、对准、键合、应力与可靠性挑战。
  • 与供应商协作材料、治具、组装流程与失效分析。

任职要求

  • 具封装、机械工程、微电子、材料、光子学或相关学科学位。
  • 具 PIC/EIC 封装、热设计、键合、测量或可靠性经验者优先。
  • 对公差、治具、流程控制与供应商沟通具良好判断力。
高级工程师 / 工程师，EDA 与光子自动化

建立连接架构、版图限制、验证与测量反馈的光子设计自动化与 EDA 流程。

薪酬

HKD 390k ~ 700k

工作内容

  • 开发 PIC 版图自动化、验证、仿真流程与数据处理工具。
  • 支持 MxPIC 式流程，使光子设计更具可重复性。
  • 与光子设计师合作，把重复任务转化为稳健软件工具。

任职要求

  • 具电气工程、光子学、计算机科学、自动化或相关学科学位。
  • 具 Python、EDA 工具、光子版图、仿真或测量自动化经验者优先。
  • 具良好软件习惯，并对实用硬件设计流程有兴趣。
项目协调员

协助整理项目数据、会议记录、排期、摘要与团队文件，让硬件项目清晰可执行。

薪酬

HKD 300k ~ 600k

工作内容

  • 整理项目文件、时程、进度摘要与行动清单。
  • 准备精简会议纪录、评审材料与跟进记录。
  • 支持工程、管理层、合作伙伴与供应商之间的沟通。

任职要求

  • 具良好组织、写作、沟通与跟进习惯。
  • 能适应技术硬件项目与工程团队环境。
  • 具清晰双语沟通能力者优先。

薪酬

HKD 10k ~ 20k/月

工作内容

  • 协助电路模块分析、仿真、文件与原型调试。
  • 审查驱动器、接收器、监测、控制或混合信号接口细节。
  • 与全职工程团队协调分析结果。

任职要求

  • 具 IC 设计、微电子或电气工程相关学习或经验。
  • 能投入结构化兼职项目工作。
  • 能清晰记录假设、结果与下一步。

薪酬

HKD 10k ~ 20k/月

  • 比较光子结构的仿真、设计意图与测量反馈。
  • 与光子及封装工程师合作完成指定项目任务。

  • 具光子学、光学工程、物理或相关领域学习或经验。
  • 熟悉硅光子、光学仿真或光子版图者优先。
  • 能谨慎记录并沟通技术假设。
兼职工程师，嵌入式电子设计

支持嵌入式电子、板级测试、固件接口与实验室验证测量自动化。

薪酬

HKD 10k ~ 20k/月

工作内容

  • 协助测试板、嵌入式接口、仪器控制与测量脚本。
  • 记录调试结果、硬件观察与可重复测试流程。
  • 与电路及验证工程师协调短周期任务。

任职要求

  • 具电子、嵌入式系统或仪器控制相关学习或经验。
  • 具实验室、固件、板级调试或 Python 自动化经验者优先。
  • 能可靠沟通兼职技术执行进度。
