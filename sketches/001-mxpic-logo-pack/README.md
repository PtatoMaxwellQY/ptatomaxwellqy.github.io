## Variant: MxPIC Logo Concept Pack

### Design stance
Six editable SVG directions for an end-to-end photonic EDA product, using OptiHK's industrial teal/green language with restrained red status accents.

### Key Choices
- Layout: A single comparison board so marks can be judged side by side.
- Typography: Technical wordmark with tight geometry and zero letter-spacing.
- Color: Foundry-teal dark mode plus a clean bright preview toggle.
- Interaction: Dark/bright preview switch for checking contrast and product fit.

### Concepts
- Waveguide Compiler: strongest primary icon candidate.
- Photonic Routing Matrix: most explicit EDA/product workflow story.
- Interferometer M: most brand-like and memorable.
- End-to-End Chip Flow: strongest platform/company presentation.
- Photonics Developer Tool: useful for CLI or automation-facing modules.
- Aperture Netlist: abstract premium direction for a mature suite.

### Trade-offs
- Strong at: Fast comparison, editable vector marks, consistency with the current OptiHK website direction.
- Weak at: Not yet production logo files; needs optical spacing, small-size testing, and final typography work.

### Best For
Choosing one or two brand directions before producing final SVG/PNG assets and website integration.
